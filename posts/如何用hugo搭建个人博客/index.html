如何用hugo搭建个人博客

May 26, 2020

如何用 hugo 搭建个人博客

  1. 从Hugo Releases 主页下载编译好的 Hugo 二进制文件: hugo_0.xxx_Windows-64bit.zip
  2. 在本地创建一个目录存储 Hugo 的可执行文件，示例为 D:/software/Hugo
  3. 将下载的安装包解压得到的 hugo.exe 放进 D:/software/Hugo
  4. 设置环境变量，将 D:/software/Hugo/ 添加到系统变量中的环境变量 Path 中即完成安装
  5. 安装好后,运行 hugo version 检测是否成功返回版本号

运行命令

  1. 运行hugo new site xxx.github.io-creator hugo 会自动生成这样一个目录结构：
▸ archetypes/
  ▸ content/
  ▸ layouts/
  ▸ static/
    config.toml

  1. 进入生成的 xxx.github.io-creator 目录： cd xxx.github.io-creator

  2. 创建一个主题：

    git init
git submodule add https://github.com/budparr/gohugo-theme-ananke.git themes/ananke`
`echo 'theme = "ananke"' >> config.toml

  3. 创建新的内容： hugo new posts/my-first-post.md 如果需要，编辑新创建的内容文件，它将从以下内容开始：

---
title: "My First Post"
date: 2019-03-26T08:47:11+01:00
draft: true
---

上传时，草稿 true 需改成 false

  1. 启动 Hugo 服务器 hugo server -D 复制地址 http://localhost:1313/ 在浏览器中打开，预览文章内容格式

  2. 自定义主题 在文本编辑器中打开 Hugo 的网站配置文件：config.toml：

    baseURL = "https://example.org/"
languageCode = "en-us"
title = "My New Hugo Site"
theme = "ananke"

将代码语言 en-us 改成 zh-Hans; example.org 改成你的 GitHub 域名，一般为 xxx.github.io

  1. 创建静态页面： hugo -D

  2. 进入 public 文件： 新建.gitignore 目录，添加 /public/

  3. 在 public 执行:

git init
git add
git commit
  1. 在 Github 上建立 xxx.github.io 远程仓库，
git remote add origin git@xxxxxxx
git push -u origin master

通关完这些步骤，你的博客就已创建完成，接下来和大家分享快乐吧。
